## 五、踩坑记录

### 1. Python 批量添加认证脚本陷阱
- **问题**: 批量添加 `Depends(get_current_user)` 脚本破坏文件结构
- **表现**: `async def func(,` 语法错误，docstring 缺失
- **解决**: 手动逐个修复或完全重写文件

### 2. psutil.disk_usage Windows bug
- **问题**: C 扩展报 `SystemError: argument 1 (impossible<bad format char>)`
- **解决**: `shutil.disk_usage()` 兜底 + 手动计算 percent

### 3. asyncpg 时区严格性
- **问题**: DB 列为 `TIMESTAMP WITHOUT TIME ZONE` 必须用 `datetime.utcnow()` (naive)
- **解决**: 全面审查 datetime 使用，区分 naive/aware 列

### 4. FastAPI 路由顺序
- **问题**: `/{dynamic}` 在 `/fixed` 之前会导致固定路径被捕获
- **解决**: 所有固定路径路由移到动态路由之前
